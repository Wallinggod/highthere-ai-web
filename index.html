Module 3

Data Science Project Lifecycle

A structured approach to solving problems with data.

1. Problem Scoping

  • What is the root cause?

2. Data Acquisition

  • Which data is important?
  • How do you source for the data?

3. Data Exploration

  • What methods can you use to clean data?

4. Modeling

  • Is this a classification or a regression problem?
  • Which of the popular AI models can you use?

5. Evaluate

  • Which metrics can you use to measure model performance?

6. Deployment

  • How can you deploy the model?
  • Predict results in real-time or consolidate daily?
